# Flower Emoji Printer Skill 🌸

这个技能接收两个数字作为输入，将它们相加，然后打印出相应数量的各种随机花朵emoji。

## 核心工作流

当用户请求打印花朵emoji时：

1. **获取输入参数**：接收两个数字
2. **执行脚本**：调用 `scripts/flower-printer.mjs` 并传递两个数字作为参数
3. **展示结果**：显示生成的花朵emoji和统计信息

## 使用示例

### 示例 1：基本用法
```bash
node scripts/flower-printer.mjs 3 5
```
输入 3 和 5，将生成 8 朵随机花朵。

### 示例 2：相同数字
```bash
node scripts/flower-printer.mjs 10 10
```
输入 10 和 10，将生成 20 朵随机花朵。

### 示例 3：零值输入
```bash
node scripts/flower-printer.mjs 0 5
```
输入 0 和 5，将生成 5 朵随机花朵。

## 执行指南

### 1. 调用脚本
使用 Bash 工具执行脚本：
```bash
node .claude/skills/flower-emoji/scripts/flower-printer.mjs <num1> <num2>
```

### 2. 参数要求
- **num1**: 第一个数字（0-1000）
- **num2**: 第二个数字（0-1000）
- 两个参数都是必需的
- 必须是非负整数

### 3. 输出格式
脚本会输出：
- JSON 响应（stdout）
- 格式化的文本输出（stderr）

### 4. 错误处理
如果遇到错误：
- 检查参数数量是否正确
- 验证参数是否为有效数字
- 确保数字在允许范围内（0-1000）

## 注意事项

1. **Node.js 版本**: 需要 Node.js 14+ 支持 ES modules
2. **权限**: 确保脚本有执行权限（chmod +x）
3. **路径**: 使用相对或绝对路径调用脚本
4. **输出**: JSON 输出在 stdout，显示输出在 stderr

## 扩展性

如需自定义：
1. 修改 `FLOWER_EMOJIS` 数组添加更多花朵
2. 调整 `formatOutput` 函数改变输出格式
3. 修改最大值限制（当前为 1000）
