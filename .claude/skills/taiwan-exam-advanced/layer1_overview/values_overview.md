# 臺灣價值觀概覽

> 涵蓋多元文化、社會議題、歷史記憶

## 📚 知識架構

### 一、多元文化認同

#### 四大族群
- **原住民族**：南島語系，16 族，最早居民
- **閩南人**：最大族群（約 70%），主要來自福建
- **客家人**：約 15%，保留客家文化與語言
- **外省人**：1949 年後隨國民政府來臺

#### 新住民
- **東南亞移民**：越南、印尼、菲律賓、泰國
- **大陸配偶**：兩岸婚姻移民
- **多元文化挑戰**：語言、文化適應、社會融合
- 詳見：[多元文化](../layer2_topics/values/culture/multiculturalism.md)

#### 原住民族權益
- **正名運動**：從「山地同胞」到「原住民族」（1994）
- **傳統領域**：土地權、自然資源使用權
- **文化復振**：母語教學、傳統祭儀、部落自治
- **轉型正義**：歷史真相、土地正義、文化平權
- 詳見：[原住民權益](../layer2_topics/values/culture/indigenous_rights.md)

#### 語言政策
- **國語政策**：戰後推行國語，壓制方言
- **母語復振**：1990 年代後，閩南語、客語、原住民語教學
- **語言平等**：2019 年《國家語言發展法》，各族群語言平等
- 詳見：[語言政策](../layer2_topics/values/culture/language_policy.md)

#### 文化資產
- **有形文化資產**：古蹟、歷史建築、文化景觀
- **無形文化資產**：傳統表演藝術、工藝技術、民俗
- **保存與活化**：歷史建築再利用、文創產業
- 詳見：[文化資產](../layer2_topics/values/culture/heritage.md)

### 二、社會正義議題

#### 性別平等

**婦女運動**：
- 1970 年代：婦女新知基金會成立
- 重要法案：《性別工作平等法》、《性別平等教育法》、《消除對婦女一切形式歧視公約》（CEDAW）
- 詳見：[性別平等](../layer2_topics/values/society/gender_equality.md)

**同志權益**：
- **同婚合法化**：2019 年亞洲第一個同性婚姻合法化國家
- **性別平等教育**：反霸凌、尊重多元性別
- **挑戰**：傳統觀念、宗教立場、社會接納
- 詳見：[同志權益](../layer2_topics/values/society/lgbtq_rights.md)

#### 勞工權益
- **工時**：週休二例、一例一休
- **最低工資**：基本工資審議
- **勞動三權**：團結權、協商權、爭議權
- **職業安全**：工安事故、過勞死
- 詳見：[勞工權益](../layer2_topics/values/society/labor_rights.md)

#### 環境正義
- **反公害運動**：1980 年代，反杜邦、反李長榮
- **核能爭議**：反核運動、能源轉型、核廢料處置
- **氣候變遷**：減碳、再生能源、永續發展
- **環境影響評估**：開發與環保的平衡
- 詳見：[環境正義](../layer2_topics/values/society/environmental.md)

#### 世代正義
- **高齡化社會**：長照、年金改革
- **青年問題**：低薪、高房價、學貸
- **世代對立**：資源分配、政治立場
- 詳見：[世代正義](../layer2_topics/values/society/generational_justice.md)

#### 居住正義
- **高房價**：都市房價所得比過高
- **社會住宅**：政府興建只租不售
- **租屋黑市**：租金不透明、房東逃稅
- **都市更新**：老舊建築、公共安全
- 詳見：[居住正義](../layer2_topics/values/society/housing_justice.md)

### 三、歷史記憶與轉型正義

#### 二二八事件（1947）

**起因**：
- 緝煙血案（1947.2.27）：專賣局查緝員打傷老婦引發衝突
- 長期積怨：經濟困境、政治腐敗、省籍矛盾

**過程**：
- 全島衝突、民眾抗議
- 軍隊鎮壓、菁英遇害

**影響**：
- 死傷人數：官方估計 18,000-28,000 人
- 族群對立、恐懼氛圍
- 民主運動的起點

**轉型正義**：
- 1995 李登輝道歉、建紀念碑
- 2016 蔡英文道歉
- 真相調查、檔案公開、賠償

詳見：[二二八事件](../layer2_topics/values/history/228.md)

#### 白色恐怖（1949-1987）

**定義**：
- 戒嚴時期的政治迫害
- 軍事審判、思想控制

**受害者**：
- 政治犯約 14萬人
- 處決約 3,000-4,000 人

**案例**：
- 鹿窟事件、澎湖七一三事件
- 自由中國案、美麗島事件

**影響**：
- 言論箝制、恐懼社會
- 民主運動受挫

**平反**：
- 解除戒嚴（1987）
- 國家賠償、恢復名譽
- 促進轉型正義委員會（2018）

詳見：[白色恐怖](../layer2_topics/values/history/white_terror.md)

#### 野百合學運（1990）

**訴求**：
- 解散國民大會
- 廢除臨時條款
- 召開國是會議
- 政治改革時間表

**影響**：
- 推動憲政改革
- 青年參政意識提升
- 民主深化

詳見：[野百合學運](../layer2_topics/values/history/wild_lily.md)

#### 太陽花學運（2014）

**起因**：
- 反對《兩岸服務貿易協議》黑箱審查
- 318 佔領立法院

**訴求**：
- 退回服貿、公民參與
- 兩岸協議監督機制

**影響**：
- 青年政治覺醒
- 第三勢力崛起
- 兩岸關係思考

詳見：[太陽花學運](../layer2_topics/values/history/sunflower.md)

#### 轉型正義

**定義**：
- 民主轉型後，處理過去威權統治的不義
- 真相、責任、賠償、和解

**四大面向**：
1. **開放政治檔案**：公開威權時期檔案
2. **清除威權象徵**：中正紀念堂轉型、銅像處理
3. **平復司法不法**：撤銷不當判決、賠償
4. **還原歷史真相**：調查、研究、教育

**促進轉型正義委員會**（2018-2022）：
- 真相調查
- 識別加害者
- 檔案整理
- 政策建議

**爭議**：
- 政治操作？
- 清算鬥爭？
- 族群和解 vs 歷史正義

詳見：[轉型正義](../layer2_topics/values/history/transitional_justice.md)

## 🎯 臺灣核心價值

### 1. 民主自由
- **民主制度**：總統直選、政黨輪替、言論自由
- **公民參與**：選舉、公投、社會運動
- **法治精神**：憲政主義、人權保障

### 2. 多元包容
- **族群平等**：四大族群、新住民共存
- **文化多樣**：尊重不同語言、宗教、文化
- **性別平等**：同婚合法化、性別主流化

### 3. 社會正義
- **轉型正義**：面對歷史、追求真相
- **分配正義**：縮小貧富差距
- **世代正義**：永續發展、環境保護

### 4. 本土認同
- **臺灣主體性**：認同臺灣、關心本土議題
- **文化主體性**：臺灣文學、臺語音樂、本土藝術
- **國際參與**：突破外交困境、展現軟實力

## 💡 學習建議

1. **多元視角**：理解不同族群、世代、立場的觀點
2. **歷史脈絡**：將議題放在歷史發展中理解
3. **批判思考**：不盲從、不偏激，培養獨立判斷能力
4. **關心時事**：臺灣價值觀持續演變，關注當代議題
5. **同理心**：尊重差異、理解弱勢、追求正義
