# 社會領域概覽

> 涵蓋臺灣歷史、地理、公民與社會

## 📚 知識架構

### 一、臺灣歷史

**時間軸**：史前時代 → 原住民時期 → 荷西時期 → 鄭氏 → 清領 → 日治 → 戰後

#### 史前時代與原住民
- **舊石器時代**：長濱文化（距今 5 萬年）
- **新石器時代**：大坌坑文化、圓山文化
- **金屬器時代**：十三行文化（煉鐵技術）
- **南島語族**：臺灣原住民屬南島語系，向外擴散至太平洋、東南亞
- **16 族原住民**：阿美、泰雅、排灣、布農、卑南等
- 詳見：[史前與原住民](../layer2_topics/social/history/prehistoric.md)、[原住民史](../layer2_topics/social/history/indigenous.md)

#### 荷西時期（1624-1662）
- **荷蘭**：佔領南部，建熱蘭遮城、普羅民遮城，發展國際貿易
- **西班牙**：佔領北部，建聖薩爾瓦多城、聖多明哥城
- **鄭成功**：1662 年驅逐荷蘭人，建立鄭氏王朝
- 詳見：[殖民時期](../layer2_topics/social/history/colonial.md)

#### 清領時期（1683-1895）
- **消極治理**：「番界」政策，限制漢人渡臺
- **移民社會**：閩南、客家移民，「三年一反、五年一亂」
- **開港通商**：1860 年天津條約，臺灣開放通商
- **建省**：1885 年臺灣建省，劉銘傳任首任巡撫，推動現代化
- 詳見：[殖民時期](../layer2_topics/social/history/colonial.md)

#### 日治時期（1895-1945）
- **武裝抗日**：霧社事件（1930）、余清芳事件
- **基礎建設**：鐵路、公路、水利、教育
- **殖民經濟**：糖業、米業、工業化（日月潭水力發電）
- **文化啟蒙**：臺灣文化協會、臺灣民眾黨
- **皇民化運動**：1937 年後，強迫改日本姓名、說日語
- 詳見：[殖民時期](../layer2_topics/social/history/colonial.md)

#### 戰後臺灣（1945-）
- **國民政府接收**：陳儀擔任行政長官
- **二二八事件**（1947）：緝煙血案引發全島衝突，國家暴力鎮壓
- **白色恐怖**（1949-1987）：戒嚴時期，政治迫害
- **土地改革**（1949-1953）：三七五減租、公地放領、耕者有其田
- **經濟發展**：1960-1980 年代經濟奇蹟，出口導向工業化
- **民主化**：1987 解嚴、1996 總統直選、2000 政黨輪替
- 詳見：[威權時期](../layer2_topics/social/history/authoritarian.md)、[民主化](../layer2_topics/social/history/democratic.md)

### 二、臺灣地理

#### 自然地理

**地形**：
- **山脈**：中央山脈、雪山山脈、玉山山脈、阿里山山脈、海岸山脈
- **最高峰**：玉山（3,952 公尺）
- **平原**：嘉南平原、屏東平原、蘭陽平原
- **盆地**：臺北盆地、臺中盆地、埔里盆地
- 詳見：[地形地貌](../layer2_topics/social/geography/topography.md)

**氣候**：
- **氣候類型**：副熱帶季風氣候
- **季節特徵**：
  - 冬季：東北季風，北部多雨
  - 夏季：西南季風，南部多雨
- **梅雨**：5-6 月，滯留鋒帶來豐沛雨量
- **颱風**：7-9 月，平均每年 3-4 個颱風侵襲
- 詳見：[氣候水文](../layer2_topics/social/geography/climate.md)

**河川**：
- **最長河川**：濁水溪（186.4 公里）
- **特徵**：短促、湍急、豐枯水期明顯
- **水庫**：石門、曾文、日月潭

#### 人文地理

**產業發展**：
- **農業**：從糖米經濟 → 精緻農業、休閒農業
- **工業**：加工出口區（高雄、臺中）→ 科學園區（新竹、臺南、中部）
- **服務業**：占 GDP 70%，資訊科技、金融、觀光
- 詳見：[產業分布](../layer2_topics/social/geography/industry.md)

**都市發展**：
- **六都**：臺北、新北、桃園、臺中、臺南、高雄
- **都市化率**：約 80%
- **區域差異**：西部都市化，東部保留自然景觀
- 詳見：[都市發展](../layer2_topics/social/geography/urban.md)

### 三、公民與社會

#### 憲政體制

**中華民國憲法**（1947）：
- **五權憲法**：行政、立法、司法、考試、監察
- **增修條文**（1991-）：凍結國民大會、總統直選、立委減半
- 詳見：[憲法](../layer2_topics/social/civics/constitution.md)

**政府體制**：
- **總統**：任期 4 年，最多連任一次，雙首長制
- **行政院**：最高行政機關，院長由總統任命
- **立法院**：113 席，任期 4 年
- **司法院**：釋憲、審判
- 詳見：[政府體制](../layer2_topics/social/civics/government.md)

#### 基本權利

**自由權**：
- 人身自由、居住遷徙自由
- 言論自由、出版自由、集會結社自由
- 宗教信仰自由

**平等權**：
- 法律之前人人平等
- 性別平等、族群平等

**受益權**：
- 生存權、工作權、財產權
- 請願權、訴願權、訴訟權

**參政權**：
- 選舉權、罷免權、創制權、複決權

詳見：[基本權利](../layer2_topics/social/civics/rights.md)

#### 民主制度

**選舉制度**：
- **總統**：相對多數決
- **立委**：單一選區兩票制（73 區域 + 34 不分區 + 6 原住民）
- **地方首長**：直接選舉

**政黨政治**：
- 藍綠對立、第三勢力崛起
- 政黨輪替（2000、2008、2016、2024）

**公民參與**：
- 公投、罷免、審議民主
- 社會運動、NGO

詳見：[民主制度](../layer2_topics/social/civics/democracy.md)

## 🎯 答題策略

### 歷史題型
1. **時間軸題**：掌握重大事件年代與先後順序
2. **因果關係題**：理解歷史事件的前因後果
3. **史料判讀題**：分析史料來源、背景、立場
4. **歷史解釋題**：多元觀點，避免單一史觀

### 地理題型
1. **地圖判讀題**：熟悉臺灣地圖、等高線、經緯度
2. **氣候圖表題**：溫度、降雨、氣壓變化
3. **產業分布題**：理解產業與環境的關係
4. **區域特徵題**：比較不同區域的差異

### 公民題型
1. **法律題**：憲法、民法、刑法基本概念
2. **政治題**：政府體制、選舉制度、政黨政治
3. **經濟題**：供需法則、市場機制、政府角色
4. **社會題**：社會階層、文化、性別、族群
