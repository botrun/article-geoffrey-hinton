# 國文領域概覽

> 涵蓋古典文學、現代文學、修辭手法

## 📚 知識架構

### 一、古典文學

**時間軸**：先秦 → 兩漢 → 魏晉南北朝 → 唐宋 → 元明清

#### 先秦文學
- **詩經**：中國最早的詩歌總集，風雅頌三部分，賦比興三手法
  - 詳見：[詩經](../layer2_topics/chinese/classical/shijing.md)
- **楚辭**：屈原為代表，浪漫主義，《離騷》為代表作
  - 詳見：[楚辭](../layer2_topics/chinese/classical/chuci.md)
- **諸子散文**：論語、孟子、莊子、荀子、韓非子
  - 詳見：[諸子百家](../layer2_topics/chinese/classical/zhuzi.md)

#### 兩漢文學
- **漢賦**：鋪陳、誇飾，代表作有《子虛賦》、《上林賦》
- **史傳文學**：司馬遷《史記》，班固《漢書》

#### 唐代文學
- **唐詩**：黃金時期，初唐四傑 → 盛唐 → 中唐 → 晚唐
  - **李白**：浪漫主義，詩仙，《將進酒》、《靜夜思》
  - **杜甫**：寫實主義，詩聖，《春望》、《登高》
  - **白居易**：新樂府運動，《琵琶行》、《長恨歌》
  - 詳見：[唐詩](../layer2_topics/chinese/classical/tangshi.md)

#### 宋代文學
- **宋詞**：婉約派（柳永、李清照）vs 豪放派（蘇軾、辛棄疾）
  - 詳見：[宋詞](../layer2_topics/chinese/classical/songci.md)

#### 元明清文學
- **元曲**：雜劇（關漢卿《竇娥冤》）、散曲
- **明清小說**：四大奇書、四大名著
  - 詳見：[古典小說](../layer2_topics/chinese/classical/novels.md)

### 二、現代文學

**起點**：五四運動（1919）

#### 白話文學運動
- **倡導者**：胡適、陳獨秀
- **理念**：「我手寫我口」，反對文言文，提倡白話文
- **代表作**：胡適《嘗試集》（新詩）、魯迅《狂人日記》（小說）
- 詳見：[白話文學](../layer2_topics/chinese/modern/vernacular.md)

#### 現代文學流派
- **鄉土文學**：魯迅、沈從文
- **新月派**：徐志摩、聞一多（重視詩的格律）
- **左翼文學**：茅盾、老舍（關注社會現實）

#### 臺灣文學
- **日治時期**：賴和（臺灣新文學之父）、楊逵
- **戰後文學**：白先勇、余光中、鍾理和、黃春明
- **鄉土文學論戰**：1970年代，反映臺灣本土意識
- **原住民文學**：夏曼·藍波安、瓦歷斯·諾幹
- 詳見：[臺灣文學](../layer2_topics/chinese/modern/taiwan_lit.md)

### 三、修辭手法

> 修辭是語言的藝術，讓表達更生動、更有感染力

#### 譬喻類
- **比喻**：明喻、暗喻、借喻
  - 範例：「人生如夢」（明喻）、「他是一頭獅子」（暗喻）
  - 詳見：[比喻](../layer2_topics/chinese/rhetoric/metaphor.md)

- **擬人**：把事物人格化
  - 範例：「春風又綠江南岸」、「花兒在微笑」
  - 詳見：[擬人](../layer2_topics/chinese/rhetoric/personification.md)

#### 誇飾類
- **誇飾**：誇大或縮小事實
  - 範例：「白髮三千丈」、「一滴水都沒有」
  - 詳見：[誇飾](../layer2_topics/chinese/rhetoric/hyperbole.md)

#### 對比類
- **映襯**：對比突出
  - 正襯：「桃花潭水深千尺，不及汪倫送我情」
  - 反襯：「蟬噪林逾靜，鳥鳴山更幽」
  - 詳見：[映襯](../layer2_topics/chinese/rhetoric/contrast.md)

#### 排列類
- **排比**：三個以上結構相似的句子
  - 範例：「為天地立心，為生民立命，為往聖繼絕學，為萬世開太平」
  - 詳見：[排比](../layer2_topics/chinese/rhetoric/parallelism.md)

- **對偶**：兩個結構相同、字數相等的句子
  - 範例：「兩個黃鸝鳴翠柳，一行白鷺上青天」
  - 詳見：[對偶](../layer2_topics/chinese/rhetoric/antithesis.md)

#### 摹寫類
- **摹寫**：模擬感官感受（視覺、聽覺、嗅覺、味覺、觸覺）
  - 範例：「春眠不覺曉，處處聞啼鳥」（聽覺摹寫）
  - 詳見：[摹寫](../layer2_topics/chinese/rhetoric/imagery.md)

#### 轉化類
- **轉化**：改變事物原本性質
  - 詳見：[轉化](../layer2_topics/chinese/rhetoric/transformation.md)

## 🎯 答題策略

### 文言文題型
1. **字詞解釋**：注意古今異義、一詞多義、通假字
2. **斷句**：找主詞、動詞、語氣詞
3. **翻譯**：直譯為主，意譯為輔
4. **主旨理解**：抓住核心概念，注意作者情感

### 閱讀理解題型
1. **主旨題**：找首尾段、關鍵句
2. **細節題**：回文定位，仔細比對
3. **推論題**：基於原文，合理推論
4. **修辭題**：辨識手法，分析效果

### 寫作題型
1. **審題**：理解題意，確定主題
2. **立意**：明確觀點，深度思考
3. **選材**：生活經驗、歷史事例、文學作品
4. **結構**：起承轉合，層次分明
5. **修辭**：適度運用，增添文采
